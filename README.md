# Atividade: Konv Mini Bank

## Objetivo:

### Essa atividade visa criar uma aplicação que simula operações financeiras de saque, depósito, apresentação de extrato/saldo e deve conter validação de CPF para evitar a duplicidade do cadastro de clientes. Dado isso, sua composição consiste em apresentação de menu ao usuário, para que escolha qual opção deseja utilizar no sistema, que por sua vez promoverá um redirecionando para o menu inicial sempre que finalizar execução de uma tarefa.

## Tecnologias Utilizadas:

### - Python;

### - CSV;

## Instruções de Uso:

### 1 - Clone o repositório com o comando : git clone https://github.com/lucianonevesln/python-konv.git;

### 2 - Se estiver usando o VSCode, execute o arquivo Python com o botão verde de 'run', ou digite 'python main.py' em console. Se preferir, aplique o passo a seguir ou vá direto para o passo 5;

### 3 - Execute o comando 'python' em console;

### 4 - Agora, insira o comando 'from main import *';

### 5 - Escolha um número conforme indicado em menu;

### 6 - Ao finalizar a utilização do sistema, escolha a opção 6 do menu para sair;

## OBS:

### - Jamais altere os arquivos de banco de dados, mantenha o cursor sempre na linha imediatamente abaixo dos metadados (cabeçalhos);

### - Iniciei a aplicação inserindo novos clientes através do menu 1;
